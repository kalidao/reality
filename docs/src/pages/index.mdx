## Install

```bash
yarn add reality
```

## Set Up

Wrap the root of your app in a [`ThemeProvider`](/components/ThemeProvider), import the external stylesheet, and add [Inter](https://rsms.me/inter/) and [iA Writer Mono](https://github.com/iaolo/iA-Fonts/tree/master/iA%20Writer%20Mono) to your font stack. (Tip: Both fonts are available on [Fontsource](https://fontsource.org).)

```tsx
import { ThemeProvider } from 'reality'
import 'reality/styles'

const App = () => {
  return <ThemeProvider>{children}</ThemeProvider>
}
```
